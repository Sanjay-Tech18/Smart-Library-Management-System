# 📚 Smart Library Management System

A Python-based **Library Management System** that allows users to issue and return books while tracking student details, issue dates, return duration, and late fines.

## 🚀 Features

✅ Dictionary-based Book Records  
✅ Issue Books with Student Name  
✅ Issue Date & Return Duration Tracking  
✅ Automatic Due Date Calculation  
✅ Progressive Fine System for Late Returns  
✅ Clean Terminal UI  
✅ Object-Oriented Programming (OOP)

---

## 🎯 Functionalities

### View Books
Displays all books and availability status.

### Issue Book
- Enter book name
- Enter student name
- Select duration

### Return Book
- Checks delay automatically
- Calculates fine if returned late

---

## 💰 Fine Policy

Late fine increases weekly:

| Week Late | Fine |
|------------|------|
| 1 Week | ₹20 |
| 2 Weeks | ₹60 |
| 3 Weeks | ₹120 |
